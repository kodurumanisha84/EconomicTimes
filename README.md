# SmartCost AI 💡

**"From Cost Detection to Autonomous Action"**

SmartCost AI is a full-stack AI-powered web application that helps businesses identify unnecessary costs and take smart actions to reduce them.

This system doesn’t just analyze — it acts.

---

## 🚀 What this project does

In this project, I built a system where a user can enter cost-related data, and the application:

* Detects inefficiencies automatically
* Explains why the issue exists
* Suggests actions to fix it
* Simulates execution
* Shows how much money can be saved

---

## 🧠 Features

* Dynamic input system (add/remove rows)
* AI-based cost analysis
* Duplicate transaction detection
* Subscription usage optimization
* Vendor pricing comparison
* SLA delay tracking
* Action execution simulation
* Savings calculation (₹)
* Basic AI chatbot for insights

---

## 🛠️ Tech Stack

* Frontend: React (Vite)
* Styling: Basic CSS / Inline styles
* Logic: JavaScript (AI simulation logic)

## 📂 Project Setup

1. Clone the repository

```bash
git clone <your-repo-link>
cd smartcost-ai
```

2. Install dependencies

```bash
npm install
```

3. Run the project

```bash
npm run dev
```

4. Open in browser

```
http://localhost:5173
```

---

## 📊 How it works

User enters data → Clicks "Analyze with AI" → System generates:

* Insights
* Actions
* Savings

---

## ⚙️ AI Logic Used

The system detects:

* Duplicate transactions (same vendor + amount)
* Underutilized subscriptions (active usage < 10%)
* Overpriced vendors (current price > market price)
* SLA delays (penalty risks)

---

## 💰 Output

For each issue, the system provides:

* Issue name
* Reason
* Confidence score
* Financial impact (₹)
